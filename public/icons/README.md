# PWA Icons for OrbitFlow CRM

This directory should contain the following icon files for the Progressive Web App:

## Required Icons

### Standard Icons
- `icon-192.png` - 192x192px - Standard app icon
- `icon-512.png` - 512x512px - Large app icon
- `icon-maskable-512.png` - 512x512px - Maskable icon (with safe zone padding)

### Apple/iOS Icons
- `apple-touch-icon.png` - 180x180px - iOS home screen icon

### Favicon
- `favicon.ico` - Multi-size ICO file (16x16, 32x32, 48x48)
- `favicon-16x16.png` - 16x16px
- `favicon-32x32.png` - 32x32px

## Design Specifications

**Brand Colors:**
- Primary: #6366f1 (Indigo/Purple)
- Background: #0f172a (Dark blue)
- Accent: Use gradient from purple to blue

**Logo:**
- Stylized "O" letter representing OrbitFlow
- Modern, clean design
- High contrast for visibility

**Maskable Icon:**
- Add 10% padding on all sides for the safe zone
- Ensure the logo is centered and doesn't get cut off when masked

## Tools for Generation

- Use tools like [RealFaviconGenerator](https://realfavicongenerator.net/) or [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- Ensure all icons are optimized for web
